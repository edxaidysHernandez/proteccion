# Prueba Técnica — Científico de Datos para Soluciones con IA
**Edxaidys Hernández Yanes**

Este repositorio contiene la solución completa a la prueba técnica para el equipo de Canales Digitales de Talento Protección.

La solución cubre los 7 puntos solicitados: análisis de productos más vendidos, identificación de causas de insatisfacción mediante clasificación con LLM, segmentación de clientes con RFM + K-Means, diseño de arquitectura de motor de recomendación, análisis geoespacial para ubicación de tienda física, construcción de base de conocimiento para agente de IA y diseño de system prompts hiperpersonalizados.

Los resultados, metodología y conclusiones de cada punto se encuentra en `Solucion_Prueba_Edxaidys_Hernandez.pdf`.
