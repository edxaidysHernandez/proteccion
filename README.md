# Prueba Técnica — Científico de Datos para Soluciones con IA
**Edxaidys Hernández Yanes**

Este repositorio contiene la solución completa a la prueba técnica para el equipo de Canales Digitales de Talento Protección.

La solución cubre los 7 puntos solicitados: 1) análisis de productos más vendidos, 2) identificación de causas de insatisfacción del cliente, 3) segmentación de clientes con RFM + K-Means, 4) diseño de arquitectura de motor de recomendación, 5) análisis geoespacial para ubicación de tienda física, 6) construcción de base de conocimiento para agente de IA y 7) diseño de system prompts personalizados.

Los resultados, metodología y conclusiones de cada punto se encuentra en `Solucion_Prueba_Edxaidys_Hernandez.pdf`.
